在使用 CardEcho 全球支付卡订阅海外服务时，如果您不打算继续使用某个软件（如 ChatGPT Plus 或 MidJourney），**请务必及时主动取消订阅**。

很多用户以为“只要卡里没钱就没事了”，这是一个严重的误区！
* **平台惩罚**：OpenAI 等商户如果连续扣款失败，可能会将您的账号列入黑名单，影响您后续绑定新卡。
* **拒付记录**：频繁的扣款失败（Card Declined）会损害卡段的信誉，为了维护 CardEcho 所有用户的用卡环境，我们可能会对产生大量恶意拒付的账户进行风控限制。

为了帮大家省钱并保护账号信用，我们整理了目前主流 AI 和流媒体服务的**保姆级取消教程**。

---

## 1. ChatGPT Plus 取消教程

OpenAI 目前有“追款机制”：如果您不主动取消，系统可能会连续 7 天尝试扣款。为了避免产生多笔拒付记录，请按以下步骤操作。

### 网页端订阅用户
1.  登录 ChatGPT 官网，点击左下角的**头像/邮箱**。
![ChatGPT 网页端取消步骤](/images/cancel-chatgpt-step1.webp)
2.  在弹出的菜单中点击 **"My plan"**。
![ChatGPT 网页端取消步骤](/images/cancel-chatgpt-step2.png)
3.  点击 **"Manage my subscription"**。
![ChatGPT 网页端取消步骤](/images/cancel-chatgpt-step3.webp)
4.  在跳转的 Stripe 页面中，点击 **"Cancel plan"** 即可。
![ChatGPT 网页端取消步骤](/images/cancel-chatgpt-step4.webp)

> **💡 提示：** 取消后，您的 Plus 会员权益**不会立即消失**，而是会一直持续到当前的账单周期结束。您可以放心提前取消，不用担心亏了时长。

### App Store (iOS) 订阅用户
如果您是在 iPhone 上通过美区 Apple ID 订阅的：
1.  打开 **App Store**，点击右上角的**头像**。
![ChatGPT iOS 端取消步骤](/images/appstoresub.webp)
2.  点击 **"订阅" (Subscriptions)**。
![ChatGPT iOS 端取消步骤](/images/chatgptplusappstore.png)
3.  在列表中找到 **ChatGPT**。
4.  点击 **"取消订阅" (Cancel Subscription)**。
![ChatGPT iOS 端取消步骤](/images/applecancel3.png)

## 2. MidJourney 取消教程

MidJourney 对扣款失败的风控较严，如果产生大量拒付可能会导致 Discord 账号受限。

1.  登录 MidJourney 官网，在左侧边栏找到您的头像卡片。
![MidJourney 取消界面](/images/midjourney-cancel-button1.png)
2.  点击 **"Manage"** 按钮。
![MidJourney 取消界面](/images/midjourney-cancel-button2.png)
3.  在 Plan 详情页，点击 **"Cancel Plan"**。
![MidJourney 取消界面](/images/midjourney-cancel-button3.png)

> **💡 注意：** 如果您是刚订阅不久且未大量使用 GPU 时间，取消时有时会触发“全额退款”选项；如果是正常使用中取消，权益将保留至本周期结束。

## 3. YouTube Premium 取消教程

1.  打开 YouTube 首页，点击右上角头像，选择 **"购买内容和会员资格"**。
2.  找到 Premium 会员栏，点击 **"管理会员资格"**。
3.  点击 **"停用" (Deactivate)**。
4.  系统会挽留你，请坚定点击 **"继续取消"**。
![YouTube Premium 取消界面](/images/youtube-premium-manage1.png)

## 4. DeepL Pro 取消教程

DeepL 的取消入口比较隐蔽：
1.  登录 DeepL 官网，访问 [账号管理页面](https://www.deepl.com/zh/account)。
2.  找到 **"DeepL Pro Starter"** (或您订阅的套餐)。
3.  点击 **"取消订阅"**。

![DeepL 取消界面](/images/cancelyourdeeplsubscribe1.png)
> **💡 试用期技巧：** 如果您开通了 30 天免费试用，建议开通后**立即去取消**。这样试用期依然有效（直到 30 天结束），但能防止到期后自动扣费，是“白嫖”试用的最安全姿势。
![DeepL 取消界面](/images/deepl2.png)

## 5. Adobe 全家桶取消教程（避坑指南）

Adobe 被称为“最难取消”的服务之一，直接在网页点取消有时会报错，或者提示要收高额的“提前解约费” (Early Termination Fee)。

**最佳解决方案：找人工客服**
1.  访问 Adobe 官网的 [帮助中心](https://helpx.adobe.com/support.html)。
2.  点击右下角的聊天气泡图标。
3.  刚开始是机器人 (Virtual Assistant) 回复，不用理会。
4.  直接在对话框输入：**"I want to talk to a real person"** 或 **"Human agent please"**。
5.  连接到真人后，告诉他：`I want to cancel my subscription immediately, please help me.`
6.  通常人工客服有权限帮您免除违约金并顺利取消。
![adobe 取消界面](/images/adobecancel.png)

### 🛡️ 实在取消不掉怎么办？（千万别直接删卡！）

如果您忘记了账号密码，或者遇到流氓商家找不到取消按钮，**请千万不要试图通过“冻结”或“删除”卡片来逃避扣费！**

这是一个非常昂贵的误区。根据国际卡组织（Visa/Mastercard）的最新规则，当商家发起扣款但因卡片余额不足或状态异常（被冻结/删除）导致交易失败时，会被标记为 **“拒付” (Decline)**。

**拒付是有代价的：**
1.  **产生罚金**：每笔失败的交易，上游银行可能会向您的账户收取 **$0.20 - $0.50 美元** 的拒付处理费。如果您订阅的服务每天尝试扣款一次，一周就会产生数美元的罚金。
2.  **触发风控**：高频的拒付记录会导致您的 CardEcho 账户被系统自动判定为“高风险”，从而限制新开卡或充值功能。

**✅ 正确的补救措施：**

1.  **联系商家人工客服 (首选)**
    * 使用上文提到的 "Human agent" 话术，要求商家后台终止服务。这是唯一彻底解决问题且不花冤枉钱的方法。
2.  **联系 CardEcho 客服报备**
    * 如果您实在无法联系到商家，请及时联系我们的在线客服。
    * 告知我们具体情况，我们会协助您处理卡片状态，尽量避免因商家恶意扣款而产生的拒付费用。

> **💰 省钱金句：** 哪怕为了不被扣那几毛钱的拒付费，也请务必走正规流程取消订阅！
