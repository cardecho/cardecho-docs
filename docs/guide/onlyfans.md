## 常见报错解决

### 1. "Transaction could not be processed"
* **原因**：通常是 IP 地址风险过高，或者账单地址没填对。
* **解决**：
    1.  检查虚拟卡余额是否充足。
    2.  **更换 IP 节点**（尝试切换到其他美国城市）。
    3.  开启浏览器**无痕模式**重试。

### 2. "Age Verification Required" (需要年龄验证)
* **原因**：部分州（如德克萨斯、弗吉尼亚等）法律要求强制验龄。
* **解决**：这是由 IP 地址决定的。最简单的办法是**切换 IP 节点**到不需要验龄的地区（如加州、纽约等），刷新页面后再试，通常就能绕过。
