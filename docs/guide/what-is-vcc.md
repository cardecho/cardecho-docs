# 什么是虚拟信用卡？

虚拟信用卡（Virtual Credit Card，简称 VCC）是一种没有实体塑料卡片的在线支付工具。它拥有和普通信用卡一样的要素：
* **16位卡号**
* **有效期**
* **CVV 安全码**

## 它有什么用？
1.  **订阅海外服务**：轻松支付 ChatGPT Plus, Midjourney, Netflix 等。
2.  **保护隐私**：在不信任的网站购物时，避免泄露真实信用卡信息。
3.  **控制预算**：充多少用多少，避免被自动扣费。

## 和支付宝/微信的区别
支付宝和微信是“钱包”，而虚拟信用卡是“卡”。很多海外商家（如 OpenAI）不支持支付宝，但支持 VISA/MasterCard 信用卡，这时就需要虚拟信用卡登场了。
