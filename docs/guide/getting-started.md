# 新手入门：什么是虚拟信用卡？

虚拟信用卡（Virtual Credit Card, VCC）是一种没有实体卡片，仅提供卡号、有效期和安全码（CVV）的在线支付工具。

## 为什么需要虚拟信用卡？

* **订阅海外服务**：支持 ChatGPT Plus、Midjourney、Netflix 等。
* **保护隐私**：不在陌生网站留下真实信用卡信息。
* **资金安全**：用多少充多少，避免被恶意扣费。
