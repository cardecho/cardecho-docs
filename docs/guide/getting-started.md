# 新手入门：什么是虚拟信用卡？

虚拟信用卡，也可以叫做数字信用卡，没有实体卡片，没有信用额度，只有卡号、过期时间和CVV（CVC），一般需要充值后才能使用，可以像实体卡一样在网上消费，部分虚拟信用卡还可以添加到Apple Pay、Google Pay使用。

## 为什么需要虚拟信用卡？

* **订阅海外服务**：支持 ChatGPT Plus、Midjourney、Netflix 、Claude、Gemini等。
* **保护隐私**：不在陌生网站留下真实信用卡信息，防止被盗刷。
* **资金安全**：用多少充多少，避免被恶意扣费。
